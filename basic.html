HTML TAGS

TAG is a keyword enclosed in angle brackets(< and >) that defines an element or structure within a webpage , used to formate and organize content.

Types of Tags

  1. Block-Level
  2. Inline

**BLOCK**

  1. Start on New Line
  2. Occupy Full Width
  3. Can Contain Inline Elements

**INLINE**

  1. Do not start on a new line
  2. Take up only as much space as needed
  3. Cannot contain block elements
